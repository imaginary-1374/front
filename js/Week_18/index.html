/RegExp/i => case-sensetive

/RegExp/g => global

/RegExp/m => multilines

/RegExp/s => one line

/x|y/ => x or y

/[0-9]/ => 0 to 9

/[^0-9]/ => not 0 to 9

(note) caps is the opposite

\w => letters and digits a-z|A-Z\0-9

\d => digits 0-9

\s => spaces

/\bword/ => matches at the beginning of a word

/word\b/ => matches at the end of a word

+ => one or more

* => zero or more

? => zero or one

{x} => x numbers

{x,y} => range betwean x,y

{x,} => at least x

$ => ends with somthing

^ => starts with somthing

?= => followed by somthing

?! => not followed by somthing
